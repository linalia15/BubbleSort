Анимированная сортировка пузырьком

Страница графически отображает процесс сортировки пузырьком целых чисел.

Как использовать?

  1. Выберете количество элементов массива (N).
  2. Cоздайте массив - нажмите на кнопку "Сгенерировать массив". Программа создаст массив из N случайных чисел от 0 до 99 и отобразит его с помощью векторной графики.
  3. Начините сортировку!

По окончанию сортировки или если нужен другой массив нажмите "Очисить".
